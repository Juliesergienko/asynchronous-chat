В архиве: 
1. chat.html - front-end сайта.
2. db.sql - файл, содержащий структуру базы.
3. chat_server.py - back-end сайта.
4. db_config.py - файл с настройками подключения к базе данных.
5. ball - папка со стилями bootstrap.
6. readme :)

Чат написан с использованием Tornado. Работает с базой PostgreSQL.
Для тестирования чата:
- инициализируйте базу из файла db.sql (при необходимости поменяйте имя пользователя user1 и название базы testchat)
- заполните соответствующие значения в файле db_config.py
- запустите chat_server.py
- перейдите на http://localhost:8080/ в браузере.
